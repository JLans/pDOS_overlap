The PDOS_OVERLAP class¶

class PDOS_OVERLAP(GAS_PDOS, REFERENCE_PDOS, adsorbate_indices, site_indices, min_occupation=0.9, sum_density=False, sum_spin=True)[source]¶

Class for calculating adsorbate-surface relative orbital overlap

Parameters

  • GAS_PDOS (vasp_dos.VASP_DOS) – VASP_DOS object of gas phase calculation of adsorbate

  • REFERENCE_PDOS (vasp_dos.VASP_DOS) – VASP_DOS object of reference adsorbate+surface system

  • adsorbate_indices (list[int]) – index (indices) of adsorbate atom(s) in REFERENCE_PDOS

  • site_indices (list[int]) – index (indices) of adsorbate atom(s) in REFERENCE_PDOS

Variables

emax (float) – maximum energy level

Notes

GAS_PDOS is used only to determine the number of orbitals that can interact with the surface and to calculate relative orbital overlap with projected density of adsorption sites without adsorbates.
