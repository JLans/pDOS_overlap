vasp_dos¶

Methods for calculating catalyst band overlap with adsorbate molecular orbitals

This is documentation for https://github.com/JLans/vasp_dos. vasp_dos enables projected density states analysis from VASP DOSCAR file

Documentation¶

See our documentation page for examples, equations used, and docstrings.

Developer¶

Dependencies¶
